[TMUX] New Window: Create a new window in the current path + Leader + t (or Alt + c)
[TMUX] Vertical Split: Create a side-by-side pane + Leader + d (or Alt + \)
[TMUX] Horizontal Split: Create a stacked pane + Leader + D (or Alt + -)
[TMUX] Close Pane: Kill the current active pane + Leader + x (or Alt + w)
[TMUX] Zoom Pane: Toggle current pane to full screen + Leader + z
[TMUX] Copy Mode: Enter vi-mode to scroll and select text + Leader + v
[TMUX] Command Palette: Fuzzy find all tmux commands + Leader + /
[TMUX] Break Pane: Move current pane to its own window + Leader + !
[TMUX] Swap Pane: Reorder panes by swapping with neighbor + Leader + > or <
[TMUX] Resize Panes: Grow or shrink panes by 5 cells + Alt + Arrow Keys
[TMUX] Quick Switch: Toggle back to the last active session + Leader + Tab
