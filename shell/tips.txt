# Keep tips short and actionable. Lines starting with # are ignored.

Tmux: Leader+Space opens the sesh session picker.
Tmux: Leader+f triggers tmux-fingers quick copy; Leader+v enters copy mode.
Tmux: new window is Leader+t or Alt+c.
Reload tmux config anytime with: tmux source-file ~/.tmux.conf
Install shell modules with: bb setup shell && bb setup tmux
Show a new tip on demand with: bb tip
See what you installed with: bb status
Update dotfiles with: bb update
Create and enter a folder quickly with: mkcd <dir>
Extract archives with: extract <file>
Link AI agent configs into a repo with: agent-link all
